# Secure Password Manager

A Python-based security application focused on robust user authentication and cryptographic data protection.

## 🔑 Key Features
* **Encryption/Decryption**: Implements secure data handling using modern crypto libraries.
* **User Vault**: A graphical interface for managing secure credentials.
* **Security Modules**: Dedicated 'src' folder containing core authentication logic.

## 🛠️ Installation
1. Clone the repository.
2. Ensure you have the `src` folder in the root directory.
3. Run `python Vault_GUI.py`.
